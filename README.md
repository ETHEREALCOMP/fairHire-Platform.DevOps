## Запуск бази даних
1. Відкрийте термінал та перейдіть у папку fairHire-Platform.Banckend
   ```
   cd <ваш_шлях_до_папки_проекту>
   ```
2. Виконайте команди Docker:

   Якщо вона була хоча б раз запущена(очищає стару):
   ```
   docker-compose down -v
   ```
   Запуск бази даних:
   ```
   docker-compose up -d
   ```
